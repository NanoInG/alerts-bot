## Що це за дичка? 🤔

Це Telegram бот для моніторингу повітряних тривог в Україні. Він:
- 📡 Перевіряє статус тривог кожні 30 секунд
- 🔔 Відправляє сповіщення в ТГ (швидше, ніж ти встигнеш одягнути труси)
- 🖥️ Має крутий tray індикатор для Windows (бо хто ж дивиться в телефон 24/7?)
- 📊 Веде історію всіх тривог (для тих, хто любить статистику)
- 🌤️ Показує погоду (бо треба знати, чи брати парасольку в бомбосховище)

## Фічі, які роблять життя кращим ✨

| Фіча | Опис |
|------|------|
| 🔴 Миттєві сповіщення | Отримуй алерти раніше за сусіда |
| 📍 Вибір регіону | Підписуйся на свою область чи район |
| 🔄 Авто-оновлення | /status оновлюється сам кожні 45 сек |
| 🚫 Rate limit | Захист від спамерів (і від тебе, коли ти панікуєш) |
| 🖥️ Tray індикатор | Зелений = чіллі, Червоний = біжи |
| ☠️ **Моніторинг русні** | **Глобальні загрози + детальний список областей.** Показуємо де саме "сатана пече млинці" і чим саме нас намагаються лякати (Ту-95, балістика тощо). Бо ворога треба знати в лице, навіть якщо воно гниле. |

## Швидкий старт 🚀

```bash
# Клонуй репо
git clone https://github.com/NanoInG/alerts-bot.git
cd alerts-bot

# Встанови залежності
npm install

# Налаштуй .env (скопіюй з .env.example і заповни)
cp .env.example .env

# Ініціалізуй базу даних
node init_db.js

# Запускай!
npm start
```

## Конфігурація 🛠️

Створи `.env` файл:

```env
# Telegram
TELEGRAM_BOT_TOKEN=твій_токен_від_бота
TELEGRAM_CHAT_IDS=твій_chat_id,-100твоя_група

# API ключі
ALERTS_API_TOKEN=токен_від_alerts.in.ua
OPENWEATHERMAP_API_KEY=твій_ключ_від_погоди

# База даних (MariaDB/MySQL)
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=твій_пароль
DB_NAME=alerts_bot
```

## Команди бота 🤖

| Команда | Що робить |
|---------|------------|
| `/start` | Привітання (бот каже "Слава Україні!") |
| `/subscribe` | Підписатись на сповіщення |
| `/unsubscribe` | Відписатись (але навіщо?) |
| `/region` | Обрати область |
| `/city` | Обрати місто/район |
| `/status` | Статус тривог + кнопка оновлення |

## Tray індикатор 🖥️

Запускається автоматично з сервером. Показує:
- 🟢 **Зелений** - все спокійно, чілль
- 🔴 **Червоний** - ТРИВОГА! Забирай кота і в укриття!
- 🟠 **Помаранчевий** - API не відповідає (можливо, інтернет впав)

## HTTP API 🌐

```
GET /api/status/:uid     - Статус для локації
GET /api/country         - Статус по всій країні  
GET /api/history         - Історія тривог
GET /history.html        - Красива сторінка історії
```

## Технології 💻

- **Node.js** - бо швидкий як тривога
- **Express** - для API
- **node-telegram-bot-api** - для Telegram
- **MariaDB** - для збереження історії
- **PowerShell** - для tray індикатора (Windows gang 💪)
